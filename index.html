Below you can find some info about the Indesign scripts in the Novelty repository.

Align to Grid

Run this script to align the selected Rectangles or TextFrames to the Document Grid.

Baseline Fitter

Sets baseline to fit page based on selected selected text leading. With the ability to adjusts page margins, styles and grid accordingly.

Change Case With Style

Easy change case of text, using regular expressions styles or both! You should thank Thomas Silkjær for this one.

Clear Paragraph Overrides

With this script you can choose to clear all paragraph overrides in the document, or only clear overrides of selected paragraph styles.

Document Frames to Bleed

This script will expand or contract rectangles (picture frames) to the bleed of the parent page. To fit the selected frame only see Frame to Bleed below.

Export Book

Exports all stories in an InDesign document or book. It will only run the book script if no InDesign documents are open. This script is based on the awesome ExportAllText.jsx from Olav Kvern.

Find Master

Go to the next page the selected master is applied to. Save preferences functionality powered by Scott Zanelli (Thank you!) Keyboard shortcut preference ctrl-M

Fore-Edge Bleeder

Place selected files in outside margin to create a fore-edge bleed. A fore-edge bleed is similar to a fore-edge painting but in this case the image is created by printing it on the die and will become visible when the book-block is cut.

Frame to Bleed

This script will expand or contract the selected rectangle (picture frame) to the bleed of the parent page. To fit all the frames in your document see Document Frames to Bleed above.

Frame to Margin

This script will expand or contract the selected frame (Text frame) to the margin of the parent page.

Get DPI Based Dimensions

This script safes the actual file dimensions (in mm) of selected image files in Adobe Bridge to a text-file on given location. Just select some files in Adobe Bridge and run the script.

Glyph Replacer

With this script you can replace the characters of a font with pictures. It is the closest thing to Photofont we have today. The script needs a folder containing psd files (glyphs) that carry the unicode decimal number of the character it represent. For example: 65.psd would link to a Capital A. Every font folder needs at least the unicode replacement character: 65533.psd which will be used when a character is not found. When you replace characters with pictures keep in mind that the script does not know anything about hyphenation... You do however keep the ability to kern, track or change the leading.

If you are looking for decimal codepoints, I love using UnicodeChecker.

Grid Division

Returns value of grid division as edit-text for you to copy.

Margin to Grid

This script aligns the margins to the grid. note this only works with grids that are a division of pagesize

Margin to Style

A script to apply the selected master-page to any page that contains the selected paragraph style.

Multi Page Importer

This awesome script from Scott Zanelli is the best way to import PDFs or InDesign natives into InDesign. This tool is similar to Place Files On Pages below

Optical Margin Batch

Apply optical margin to all stories in the document

Orphan Fixer

Avoid orphans by applying a no-break style to the last space in paragraphs.

Pair Paragraph Styles

Fix paragraph style combinations with this tool by Thomas Silkjær

Place Files On Pages

A comic book helper. It places pages as Image or PDF into a book, you can choose to size the art and scale down to bleed, page or margin. If you need more controle of placing PDFs or InDesign files Multi Page Importer above is really good.

Read Actual Margins

The margins window showes three decimal places. Under the hood however the margins could be set (by a script) to way more decimal pages. This script shows the actual unrounded margins with the option to round them. This should fix the ‘Unable to create boundingbox’ error.

Remove Frame Rotation

Moves the rotation of frames to it’s contents. This way you end up with rotated content in stead of rotated frames.

Remove All Guides

A small tool to clear all guides in one go.

Replace Master

Swap one master-page for another.
